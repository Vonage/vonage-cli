# Usage

<!-- usage -->
```sh-session
$ npm install -g @vonage/cli-plugin-users
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
@vonage/cli-plugin-users/1.1.2 linux-x64 node-v16.14.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
